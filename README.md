# 8086-Assembly-Digital-Clock
#Proje Amacı 8086 mikroişlemci mimarisi kullanılarak geliştirilen, sistem saatini okuyup ekrana yazdıran düşük seviyeli programlama projesidir. 
#Kullanılan Teknolojiler  
Dil: 8086 Assembly 
Emülatör: Emu8086 
Kritik Konseptler: Interrupt kullanımı (INT 21h, INT 10h). Register yönetimi (AX, BX, CX, DX). Bellek adresleme (Memory Addressing).
